# Strive - React + Tailwind CSS

A React.js recreation of the [Strive Bootstrap Business Template](https://bootstrapmade.com/demo/Strive/) by BootstrapMade, built with Vite, React, and Tailwind CSS v4.

## Sections

- Header with top bar & navigation
- Hero carousel slider
- About
- Services & Features
- CTA with stats
- Client logos
- Portfolio with filters
- Pricing
- FAQ accordion
- Team
- Blog
- Contact form & map
- Footer with newsletter

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for Production

```bash
npm run build
npm run preview
```

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Lucide React (icons)
